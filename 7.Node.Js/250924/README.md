## 오늘의 수업 핵심

> **"테이블 간의 관계를 표현하는 법을 익힌다."**

테이블을 만들고 끝나는 게 아니라,  
관계를 정의하고,  
그 관계를 기반으로 데이터를 정확히 넣고,  
원하는 방식으로 꺼낼 수 있어야 한다.

**관계형 데이터베이스(Relational Database)**
